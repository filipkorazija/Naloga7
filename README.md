# Projekt – Roubet

## Glavne funkcionalnosti

1. **Uporabniška prijava in registracija**  
   Sistem omogoča ustvarjanje računa in prijavo registriranih uporabnikov.

2. **Iskanje vsebin**  
   Uporabnik lahko prek iskalne vrstice išče vsebine po ključnih besedah.

3. **Urejanje uporabniškega profila**  
   Uporabnik lahko spremeni osebne podatke, geslo in profilno sliko.

4. **Kontaktni obrazec**  
   Uporabniki lahko pošljejo sporočilo administratorju aplikacije.

5. **Prikaz domače strani z osnovnimi informacijami**  
   Aplikacija vsebuje domačo stran z uvodnimi informacijami o projektu.
